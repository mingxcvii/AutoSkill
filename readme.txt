AutoSkill : Version 1.0

[TH] วิธีใช้งาน
  1. เปิด AutoSkill.exe
  2. เลือกหน้าต่างเกมจาก "Target Game Window"
  3. ติ๊ก Active เพื่อเปิดสกิล, คลิกปุ่ม Hotkey เพื่อเปลี่ยนปุ่ม, ลาก Cooldown ตั้งเวลาหน่วง
  4. กดปุ่ม Toggle (ค่าเริ่มต้น: HOME) เพื่อเปิด/ปิด Macro
  5. กด INSERT เพื่อซ่อน/แสดงหน้าต่าง
  6. กด ESC ขณะตั้งปุ่มเพื่อยกเลิก

[EN] How to Use
  1. Run AutoSkill.exe
  2. Select your game window from "Target Game Window"
  3. Check Active to enable a skill, click Hotkey button to rebind, drag Cooldown to set delay
  4. Press Toggle key (default: HOME) to start/stop the macro
  5. Press INSERT to hide/show the window
  6. Press ESC while binding a key to cancel
