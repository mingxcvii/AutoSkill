AutoSkill : Version 1.1

[TH] วิธีใช้งาน
  1. เปิดโปรแกรม AutoSkill
  2. การใช้งานหลายจอ (Multi-boxing):
     - กดปุ่ม "New" แล้วตั้งชื่อ Profile (เช่น Preset1) จากนั้นกด Add
     - แนะนำให้เปิดเกมทีละจอ แล้วเลือกหน้าต่างเกมจากช่อง "Target Game Window" (จะมีเลข PID ขึ้นหน้าชื่อเกมเพื่อให้แยกจอได้ง่ายขึ้น)
     - ตั้งปุ่ม "Enable Macro" ของแต่ละจอให้ต่างกัน (เช่น จอ 1 ใช้ MOUSE4, จอ 2 ใช้ MOUSE5) เพื่อสั่งรันมาโครแยกกันได้อย่างอิสระ
  3. ตั้งค่าสกิล: 
     - ติ๊กถูกที่ Active เพื่อเปิดใช้งานสกิลนั้นๆ (ปุ่มไหนไม่ได้ใช้ ไม่ต้องติ๊ก)
     - คลิกที่กล่องปุ่มเพื่อตั้งปุ่มคีย์บอร์ดที่ต้องการ
     - ดับเบิ้ลคลิกที่ช่อง Cooldown เพื่อตั้งค่าหน่วงเวลา (ถ้าใช้แบบเดิมๆ ก็ไม่ต้องแก้)
  4. กดปุ่ม Enable Macro ที่ตั้งไว้ เพื่อเริ่ม/หยุด การทำงาน (โปรแกรมจะสลับไปยังหน้าจอเกมของจอนั้นให้อัตโนมัติ)
  5. กดปุ่มซ่อนเมนู (ค่าเริ่มต้น: INSERT) เพื่อซ่อนหรือแสดงหน้าต่างโปรแกรม
  6. ขณะกำลังคลิกเพื่อตั้งปุ่ม หากต้องการยกเลิกให้กด ESC

[EN] How to Use
  1. Run AutoSkill application.
  2. Multi-boxing Setup:
     - Click "New", enter a Profile name (e.g., Preset1), and click Add.
     - It is recommended to open the game clients one by one. Select the specific window from the "Target Game Window" dropdown (each window will have a unique PID number).
     - Set a unique "Enable Macro" hotkey for each profile (e.g., MOUSE4 for Client 1, MOUSE5 for Client 2) to control them independently.
  3. Skill Setup:
     - Check 'Active' to enable a specific skill. (If you don't use a skill, simply leave it unchecked).
     - Click the button box to rebind your hotkey.
     - Double-click the Cooldown box to set the delay (leave as default if no changes are needed).
  4. Press the Enable Macro hotkey you set to start/stop the macro. (Starting it will automatically focus that profile's game window)
  5. Press the Hide Menu hotkey (default: INSERT) to hide/show the main window.
  6. Press ESC while binding a key to cancel.
